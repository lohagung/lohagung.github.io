Tutorial Penggunaan

Selamat datang di aplikasi Manajemen Data RT. Berikut adalah beberapa langkah untuk memulai:

  • Klik pada menu Bendahara untuk mengelola keuangan RT.
  • Klik pada menu Jimpitan untuk mengelola jimpitan warga.
  • Klik pada menu Keamanan untuk mengelola denda ronda.
  • Klik pada menu Selokan untuk mengelola iuran dan pengeluaran selokan.
  • Klik pada menu Surat untuk mengelola surat undangan rapat.
  • Klik pada menu Arisan untuk mengelola kegiatan arisan.
  • Klik pada menu Lainnya untuk fitur lainnya akan ditambahkan menyusul..
